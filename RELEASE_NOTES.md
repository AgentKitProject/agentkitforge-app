# AgentKitForge Release Notes

## v0.1.0

Initial desktop app release for local Agent Kit workflows.

### Included

- Tauri + React + TypeScript desktop shell.
- My Kits local library for local Agent Kit folders.
- Agent Kit validation with `agentkitforge-core` profiles.
- Template-based kit creation.
- AgentKitDraft JSON rendering.
- OpenAI-assisted AgentKitDraft generation.
- Use inside Forge with OpenAI and core Context Builder.
- One-file Markdown export for web assistants.
- `.agentkit.zip` package/export and import.
- Codex skills directory export target.
- Local Settings for OpenAI key, model, output folder, validation profile, and context defaults.
- About screen with local privacy and API key storage caveats.

### Notes

- Code signing is not configured yet.
- Auto-update is not implemented.
- Agent Kit Market, org/account features, and hosted infrastructure are out of scope for v0.1.0.
- macOS and Linux release packaging are future work unless built manually in those environments.
